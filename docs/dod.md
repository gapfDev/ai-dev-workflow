# Definition of Done

## Code Ticket DoD
- Scope implemented as described in issue acceptance criteria.
- No regression in capture, menu visibility, and Kanban status flow.
- Smoke checks pass or blockers documented.

## QA Ticket DoD
- Reproducible steps and expected results are documented.
- Automated checks pass for critical workflows.
- Failures include actionable diagnosis.

## Deploy/DevOps Ticket DoD
- Rollback path is documented and tested.
- Required Script Properties are documented.
- Deployment evidence includes URL and test status.

## Docs Ticket DoD
- Content is in English.
- Links are valid and accessible from `README.md`.
- Operational owner is identified.
