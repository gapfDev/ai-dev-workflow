# PRODUCT VISION

## 1. Project Type
- **Greenfield** project (Mobile App).

## 2. Core Features & Scope
- Fitness tracking application for Android.
- Tracks gym routines and cardio.
- Authentication via Google Login.
- GPS tracking for outdoor cardio exercises.

## 3. Workflows
- User logs in with Google.
- User selects exercise type (Gym / Cardio).
- If Cardio, GPS tracking is initiated.

## 4. Uncovered Constraints & Blindspots
- Wearables integration: Ignored for MVP mapping.
- Nutrition / Social / Offline features: De-scoped currently due to lack of human confirmation. Focus solely on core gym/cardio tracking.
